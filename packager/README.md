## Overview

[**Z**ongsoft.**T**ools.**P**ackager](https://github.com/Zongsoft/tools/tree/main/packager) is a tool for creating installation packages, supporting formats such as `.tar.gz`, `.deb`, and `.rpm` for _**L**inux_ platform.

### Basic Usage

```shell
dotnet-pack tar <options...> [arguments...]
dotnet-pack deb <options...> [arguments...]
dotnet-pack rpm <options...> [arguments...]
```
