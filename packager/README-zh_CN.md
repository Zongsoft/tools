## 概述

[**Z**ongsoft.**T**ools.**P**ackager](https://github.com/Zongsoft/tools/tree/main/packager) 是一个制作安装包的工具，可提供 `.tar.gz`、`.deb`、`.rpm` 等 _**L**inux_ 平台的安装包。

### 基本用法

```shell
dotnet-pack tar <选项...> [参数...]
dotnet-pack deb <选项...> [参数...]
dotnet-pack rpm <选项...> [参数...]
```
